# Shopify Sales Customer Analysis Power BI

This project is a Power BI dashboard built to analyze Shopify sales data and generate actionable business insights.

The main goal is to:
- Track sales performance
- Understand customer behavior
- Measure customer retention & lifetime value
- Help businesses make data-driven decisions

Project Workflow:
- Requirement Gathering
- Data Understanding
- Data Cleaning & Quality Check
- Data Modeling
- DAX Calculations
- Dashboard Design
- Insights Generation

Key KPIs Used:
- Transaction Performance
  
Net Sales -> Total revenue generated

Total Quantity -> Products sold

Average Order Value (AOV) -> Revenue per order

- Customer Behavior
Total Customers
Single Order Customers
Repeat Customers
- Retention Metrics
Lifetime Value (LTV)
Repeat Rate
Purchase Frequency

Dashboard Features:
- Overview Dashboard
KPI cards for quick insights
Sales trend over time (daily)
Hourly sales pattern
- Regional Analysis
Map showing sales by province & city
Top-performing cities (bar chart)
- Customer Insights
Repeat vs new customers
Purchase behavior tracking
- Payment Analysis
Sales by payment gateway (Shopify, PayPal, etc.)
- Product Analysis
Revenue by product types (Shoes, Clothing, etc.)
- Drill-Through Detail Page
Full transaction-level data table
Helps validate insights

Key Insights:
- Certain cities contribute the highest revenue
- Repeat customers significantly impact revenue
- Specific payment gateways dominate transactions
- Some product categories outperform others
- Sales vary by time -> useful for marketing timing
